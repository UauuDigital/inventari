Enganxa aquí la URL del Google Apps Script per desar productes nous directament al full de Google.

Al full → Extensions → Apps Script, enganxa aquest codi i despliega'l com a App Web (accés: Tothom): 

function doGet(e) {
  const s = SpreadsheetApp
    .getActiveSpreadsheet().getSheets()[0];
  const p = e.parameter;
  s.appendRow([p.id, p.producte, p.proveidor,
               p.categoria, p.codi, p.min,
               p.unitat, p.unitspercaixa]);
  return ContentService.createTextOutput('ok');
}
